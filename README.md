# Diabetes Prediction using KNN

This project demonstrates the use of K-Nearest Neighbors (KNN) algorithm to predict whether a person has diabetes.

## 📌 Problem Statement
Predict diabetes based on health parameters like glucose, BMI, age, etc.

## 🔍 Concepts Used
- Data preprocessing
- Feature scaling (StandardScaler)
- Train-Test Split
- KNN Algorithm
- Hyperparameter tuning (K selection)
- Model evaluation

## 🛠 Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 📊 Dataset
Includes features:
- Pregnancies
- Glucose
- BloodPressure
- BMI
- Age

Target:
- Outcome (0 = No Diabetes, 1 = Diabetes)

## ⚙️ Workflow
1. Load dataset
2. Split into X and y
3. Apply scaling
4. Train KNN model
5. Test different K values
6. Evaluate accuracy

## 📈 Result
Model performance varies with K value. Best K selected based on highest accuracy.

## 💡 Key Learning
- Importance of feature scaling in KNN
- How K value affects model performance
- Practical implementation of classification
